Módulo Mi Legajo

Descripción del Proyecto:

Este es un módulo que se encuentra dentro la plataforma que utiliza la Municipalidad de Luján de Cuyo, en donde los agentes municipales pueden gestionar sus licencias ya sean anuales o especiales, cargar documentación, ver y descargar sus bonos de sueldos, actualizar su información personal; de manera ágil y sencilla, elimanando el uso de papel.
Son más de dos mil agentes municipales que hacen uso de este módulo a diario.


Tecnologías Utilizadas:

  • Lenguajes de Programación: PHP, JavaScript
  • Base de Datos: MySQL
  • Frontend: HTML, Bootstrap
  • Framework PHP: CodeIgniter
  • Patrón de Diseño: Modelo Vista Controlador (MVC)

Detalles del Desarrollo:

  • PHP y CodeIgniter: Utilicé PHP y el framework CodeIgniter para el desarrollo del backend, lo que permitió una gestión eficiente de las solicitudes del cliente y una estructura organizada del proyecto. CodeIgniter facilitó la implementación del patrón MVC, garantizando un código mantenible y escalable.
  • JavaScript: Implementé funcionalidades dinámicas en el frontend utilizando JavaScript para mejorar la experiencia del usuario. Esto incluyó validaciones de formularios, interacciones en tiempo real y actualizaciones de la interfaz de usuario.
  • MySQL: Diseñé y gestioné la base de datos utilizando MySQL para almacenar la información de los agentes, licencias, documentos y recibos de sueldo. Utilicé consultas SQL eficientes para acceder y manipular los datos de manera segura y rápida.
  • HTML y Bootstrap: Diseñé una interfaz de usuario intuitiva y amigable utilizando HTML y el framework Bootstrap. Garanticé una experiencia consistente y receptiva en diferentes dispositivos y tamaños de pantalla.

Resultados y Aprendizaje:

El desarrollo del módulo "Mi Legajo" para la Municipalidad de Luján de Cuyo representó un desafío emocionante que me permitió aplicar mis habilidades en desarrollo full stack para satisfacer las necesidades específicas del cliente. Aprendí a trabajar en un entorno real, comprendiendo los requisitos del usuario y traduciéndolos en una solución técnica efectiva. Además, fortalecí mis habilidades en el uso de herramientas como CodeIgniter, MySQL y Bootstrap, lo que amplió mi experiencia como desarrollador full stack.
Este proyecto no solo proporcionó una solución práctica y eficiente para la gestión de recursos humanos, sino que también me permitió crecer profesionalmente y mejorar mis capacidades en el desarrollo de aplicaciones web completas.
